Run MySQL Server on Docker - Development

Posted on 2020, Aug 02

While developing application on your machine, you don’t really need to run them on the host. So instead of that, you can rely on running them on docker. As a first step, you can use the below command to run mysql server in the background service as a daemon:

docker run --name mysql-dev -e MYSQL_ROOT_PASSWORD=password -p 3306:3306 -d mysql

verify the service:

➜ docker ps
CONTAINER ID        IMAGE                    COMMAND                  CREATED             STATUS              PORTS                               NAMES
6ed804fdd24a        mysql                    "docker-entrypoint.s…"   26 minutes ago      Up 26 minutes       0.0.0.0:3306->3306/tcp, 33060/tcp   mysql-dev

You can access mysql from your host’s termina:

mysql -uroot -p -h127.0.0.1
