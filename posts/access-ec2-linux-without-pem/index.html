Access Ec2 Linux Without Pem

Posted on 2020, Aug 10

Sometimes you don’t need to provide or share the pem key with other team members. You can simply do the following in order to avoid that:

  • Create a new user
  • Add the user to the sudeor file
  • Enable password authentication

Here’re the details:

Create the new user:

sudo adduser mynewuser

Add the user to the sudeor file:

sudo visudo

Add the following

mynewuser   ALL=(ALL:ALL) ALL

Update the ssh config file:

# /etc/ssh/sshd_config
Passwordauthentication yes

Restart ssh:

sudo /etc/init.d/ssh restart

You can now connect directly with using your password only.
