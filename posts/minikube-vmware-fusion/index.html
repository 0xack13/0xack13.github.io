Minikube can be installed and uses vmware fusion as a driver to virtualize Minikube:

To install the vmware driver:

brew install docker-machine-driver-vmware

Then, we can easily install minikube along wih starting minikube with vmware driver:

brew install minikube
minikube start --driver=vmware
